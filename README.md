# 🌐 BlogSphere  

🚀 A sleek, modern, and responsive blogging platform designed for seamless reading experiences. Built with **HTML, CSS, and JavaScript**, BlogSphere provides an elegant UI with smooth animations and a minimal design approach.  

🔗 **Live Demo:** [Click Here](https://sadik47x.github.io/vaultofcodes-blogsphere)  
📂 **GitHub Repo:** [Click Here](https://github.com/Sadik47x/vaultofcodes-blogsphere)  

---

## 📖 Description  
**BlogSphere** is a fully responsive blogging website that delivers a clean and interactive interface for users to explore and read blog posts.  
This project was built as part of my **internship at VaultofCodes**, showcasing modern UI design, responsive layouts, and smooth user interactions.  

---

## ✨ Features  
- 📱 **Responsive Design** – Works seamlessly on desktop and mobile devices.  
- 🎨 **Modern UI** – Clean, minimal, and visually appealing design.  
- ⚡ **Smooth Animations** – Hover and transition effects for better user engagement.  
- 📰 **Blog Post Cards** – Blogs displayed in a structured card layout.  
- 🔗 **Clickable Buttons** – Interactive "Read More" buttons for user navigation.  

---

## 🛠️ Tech Stack  
- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript** – Interactivity
